Inheritance->Child class or sub class inherit properties from parent class/super class ud=sing extends keyword
->Parent class always executes first
->Super keyword used to access parent class variables method constructor

Types:
1.single Inheritance A->B
2.multilevel Inheritance  A->B->C
3.hierarchy Inheritance  A->B A->C A->D Single parent multiple Child
4.multiple Inheritance  A->C B->C multiple parents single child
5.hybrid Inheritance Combination of any two Inheritance
